Detailed Description

Maintain the complete state of user interactions (mouse, keyboard) for a given view or window.

All coordinates are recaled with respect to the associated view. To check if the B key is currently held: input.key().isActivated(Key::B); To check if the right mouse was just released: input.mouseButton().isReleased(Mouse::Right);
