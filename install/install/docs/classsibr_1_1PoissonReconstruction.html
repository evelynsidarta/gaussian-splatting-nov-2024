Initialize reconstructor for a given problem.

Gradients and target are expected to be RGB32F, mask is L32F. In the mask, pixels with value = 0 are to be inpainted, value > 0.5 are pixels to be used as source/constraint, value < -0.5 are pixels to be left unchanged and unused. To compute the gradients from an image, prefer using PoissonReconstruction::computeGradients (weird results have been observed when using cv::Sobel and similar).

Parameters
gradientsXthe RGB32F horizontal color gradients to integrate along
gradientsYthe RGB32F vertical color gradients to integrate along
maskthe L32F mask denoting how each pixel should be treated.
img_targetthe RGB32 image to use as a source constraint (will be copied internally)
