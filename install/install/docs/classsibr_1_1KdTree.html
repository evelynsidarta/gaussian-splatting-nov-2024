Detailed Description

template<typename num_t = double, class Distance = nanoflann::metric_L2>
class sibr::KdTree< num_t, Distance >

Represent a 3D hierachical query structure baked by a nanoflann KdTree.

Note
With the default L2 distance, all distances and radii are expected to be the squared values (this is a nanoflann constraint). For other metrics, use the distance directly.
