Detailed Description

This class can be used to cast rays against a scene containing triangular meshes.

You can check for intersections with the geometry and get information about the hit (such as coordinates, distance, triangle id).

You should have one or few instance of this class (for performance purposes). Each instance can run in parallel.

Note
This abstraction is built on top of Embree.
Warning
There is no backface culling applied.
