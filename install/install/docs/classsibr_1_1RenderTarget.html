Detailed Description

template<typename T_Type, unsigned int T_NumComp>
class sibr::RenderTarget< T_Type, T_NumComp >

A render target wraps an OpenGL framebuffer, that can have one depth buffer, one stencil buffer, and one or more color attachments.
