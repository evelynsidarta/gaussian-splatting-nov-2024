Detailed Description

template<typename T>
class sibr::GLuniform< T >

OpenGL shader uniform wrapper with additional update/storage functions.

It will behave as an element of the type it wraps, but can also be set/sent to the GPU. This limits duplication, as you can replace a float+GLParameter by a GLuniform<float> When you need a reference to the value (for instance in imGui), use uniform.get().
