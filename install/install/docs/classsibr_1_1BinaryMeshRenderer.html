◆ getEpsilon()

float& sibr::BinaryMeshRenderer::getEpsilon ( )
inline

Shift that can be used to modify the depth written, to avoid Z-fighting when rendering multiple masks of the same mesh or combining masks.

If set to 0.0: no shift, if set to 1.0: all vertices sent to depth 0.0.

Returns
a reference to the shift
