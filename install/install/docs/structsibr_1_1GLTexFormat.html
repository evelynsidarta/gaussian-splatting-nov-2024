Detailed Description

template<typename ImageType, typename ScalarType = typename ImageType::Type, uint N = ImageType::e_NumComp>
struct sibr::GLTexFormat< ImageType, ScalarType, N >

Helper class to provide, from a generic image type, all the information needed for OpenGL textures Right now it can work with all sibr::Image and with cv::Mat (3U8 only) You can add more using explicit template instanciation to specify both ValidGLTexFormat and the following GLTexFormat properties.
