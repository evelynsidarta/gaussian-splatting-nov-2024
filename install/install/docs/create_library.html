Shaders

Shaders copy to binary folder is handled by the library CMake.
You can put your shaders files in renderer/shaders with the following extensions:

  • .vert
  • .frag
  • .geom
  • .vp
  • .fp
  • .gp

If unchanged, those shaders should be copied to install/bin/shaders_rsc.
To extend this behavior, please update renderer/CMakeLists.txt.
